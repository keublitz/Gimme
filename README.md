## How to use

1. Type in the URL of the video.
2. Click "Gimme the video" or "Gimme the audio".

That's it!

### Notice for developers regarding 0.1.0:
> This branch is currently non-functional without ```yt-dlp``` and ```ffmpeg``` binaries placed in a folder path ```~/Gimme/Binaries```. This will be fixed in a later version.

## Settings/supported formats

* Enable HDR downloads. SDR only by default.
* Set a maximum resolution for video downloads. 1080p by default
* Supported video formats: ```mp4```, ```mkv```, ```mov```, ```webm```, ```flv```, ```avi```. Set to ```mp4``` by default.
* Supported audio formats: ```mp3```, ```m4a```, ```alac```, ```flac```, ```aac```, ```wav```, ```opus```, ```vorbis```. Set to ```mp3``` by default.
